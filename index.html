The Array methode :

concat ()

Un nouveau tableau formé du tableau courant avec un ou plusieurs autres tableaux, et/ou valeurs.

Exemple : concat ()

const numb1 = [1, 2, 3];

const letter = ['a', 'b', 'c'];

const numb2 = [4, 5, 6];

const numb3 = numb1.concat(letter);

console.log(numb3);

Array(6) [ 1, 2, 3, "a", "b", "c" ]

const numb4 = numb1.concat('letter', 'food', 'Gesi');

console.log(numb4); :

Array(8) [ 1, 2, 3, "a", "b", "c", "letter", "food", "Gesi" ]

The Array methode :

join ()

Joint tous les éléments du tableau courant et les retourne en une seule chaîne avec le séparateur choisi.

Exemple : join ()

const arrUno = ["Joint", "les", "éléments", "d'un", "tableau", "avec", "le séparateur", "choisi"];

console.log(arrUno);

Array(7) [ "Joint", "les", "éléments", "d'un", "tableau", "avec", "le séparator", "choisi" ]

var chaineUno = arrUno.join('+');

console.log(chaineUno);

Joint+les+éléments+d'un+tableau+avec+le séparator+choisi

The Array methode :

sort ()

Retourne un tableau une fois les éléments du tableau courant triés (le tri par défaut est alphabéique et ascendant).

Exemple : sort ()

const upSort = ["le tri", "par défaut", "est alphabéique", "et ascendant"];

console.log(upSort);

Array(4) [ "le tri", "par défaut", "est alphabéique", "et ascendant" ]

upSort.sort();

// Attention aux majuscule / miniscule de la premiére lettre du mot

Array(4) [ "est alphabéique", "et ascendant", "le tri", "par défaut" ]

The Array methode :

toString ()

Convertit un tableau en chaîne de caractères et retourne la chaîne.

Exemple : toString ()

const arrUn = ["Convertit", "un", "tableau", "en", "chaîne", "de", "caractères"];

console.log(arrUn.toString());

Convertit,un,tableau,en,chaîne,de,caractères

console.log(arrUn);

Array(7) [ "Convertit", "un", "tableau", "en", "chaîne", "de", "caractères" ]

var arrDeux = arrUn.toString();

console.log(arrDeux);

Convertit,un,tableau,en,chaîne,de,caractères

The Array methode :

unshift ()

Retourne un nouveau tableau avec une nouvelle longueur par ajout d'un ou plusieurs éléments au début
du tableau.
(destructif)

Exemple : unshift ()

const ajout = ["Retourne", "un nouveau", "tableau", "avec une", "nouvelle longueur", "par ajout d'un ou", "plusieurs éléments"];

console.log(ajout);

Array(7) [ "Retourne", "un nouveau", "tableau", "avec une", "nouvelle longueur", "par ajout d'un ou", "plusieurs éléments" ]

ajout.unshift("ajout1", "ajout2");

console.log(ajout);

Array(9) [ "ajout1", "ajout2", "Retourne", "un nouveau", "tableau", "avec une", "nouvelle longueur", "par ajout d'un ", "ou plusieurs éléments" ]

The Array methode :

shift ()

Supprime le premier élément d'un tableau et retourne cet élément. La longueur du tableau étant modifiée en concéquence.
(destructif)

Exemple : shift ()

const leftShift = ["Supprime", "le premier élément", "d'un tableau et", "retourne cet élément"];

console.log(leftShift);

Array(4) [ "Supprime", "le premier élément", "d'un tableau et", "retourne cet élément" ]

console.log(leftShift.shift());

Supprime

console.log(leftShift);

Array(3) [ "le premier élément", "d'un tableau et", "retourne cet élément" ]

The Array methode :

pop ()

Supprime le dernier élément d'un tableau et retourne cet élément. La longueur du tableau étant modifiée en concéquence.
(destructif)

Exemple : pop ()

const arrUn = ["Supprime", "le", "dernier", "élément", "d'un", "tableau"];

console.log(arrUn);

Array(6) [ "Supprime", "le", "dernier", "élément", "d'un","tableau" ]

arrUn.pop();

console.log(arrUn);

Array(5) [ "Supprime", "le", "dernier", "élément", "d'un" ]

The Array methode :

push ()

Ajout un ou plusieurs éléments à la fin du tableau courant et retourne sa longueur à jour. La longueur du tableau étant modifiée en concéquence.
(destructif)

Exemple : push ()

const rightShift = ["Ajout un", "ou plusieurs éléments", " à la fin du tableau courant", "et retourne sa longueur", "à jour"];

rightShift.push("avant dernier élément", "dernier élément");

7

console.log(rightShift)

Array(7) [ "Ajout un", "ou plusieurs éléments", " à la fin du tableau courant", "et retourne sa longueur", "à jour", "avant dernier élément", "dernier élément" ]
